##  Brief One Line Summary
This project presents an interactive Excel dashboard to analyze Indian Premier League (IPL) data, focusing on team performance, match outcomes, and key trends.

---

##  Overview
The IPL Dashboard is designed to provide a clear and visual understanding of IPL match data using Microsoft Excel.  
It helps users quickly identify patterns such as top-performing teams, match results, and season-wise insights through charts and KPIs.

---

##  Problem Statement
IPL data is large and difficult to analyze in raw tabular form.  
The goal of this project is to transform raw IPL data into a **visual, easy-to-understand dashboard** that supports quick analysis and decision-making.
